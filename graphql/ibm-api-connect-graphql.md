# IBM API Connect GraphQL API

IBM API Connect is a comprehensive API management platform. The API covers API design, lifecycle management, developer portal, security enforcement, analytics, application registration, and integration with IBM Cloud and on-premises deployments.

**Endpoint:** No native GraphQL endpoint — conceptual schema derived from IBM API Connect REST API
**Documentation:** https://www.ibm.com/docs/en/api-connect/

- Reference: https://cloud.ibm.com/apidocs/apiconnect/apic-management-api
